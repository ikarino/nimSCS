# nimSCS
至高のプログラミング言語[Nim](https://nim-lang.org/)によるscs実装です。

## インストール
```bash
git clone https://github.com/ikarino/nimSCS.git
cd nimSCS
nimble build -d:release
```

## 使い方
```bash
bin/scs [json input file]
```

## 入力形式
[torneko3js](https://github.com/ikarino/torneko3js)と同じ
